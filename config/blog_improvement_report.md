## 2. 핵심 개선 사항 (우선순위별)

### [긴급] 1순위: 쿠팡 파트너스 수익 극대화
**현재:** 대부분의 포스트에 쿠팡 링크 없음
**목표:** 모든 신규 포스트 + 기존 인기 포스트에 상품 링크 삽입

- 포스트당 쿠팡 상품 3개 삽입 (자동화 시스템 활용)
- 건강식품, 운동용품, 뷰티 제품 중심
- H2 섹션 뒤에 자연스럽게 배치
- 고지문 필수 포함

### [긴급] 2순위: 발행 빈도 정상화
**현재:** 불규칙 (12월→2월, 20일 공백)
**목표:** 매일 1포스트 자동 발행

- 스케줄러 활성화 (매일 09:00)
- 트렌드 키워드 기반 자동 콘텐츠 생성
- 최소 주 5회 발행 유지

### [높음] 3순위: SEO 전면 개선
**제목 최적화:**
- 현재: "뼈가 약해졌다면 지금 꼭 챙기세요" (12자, 키워드 부재)
- 개선: "뼈 건강에 좋은 음식 6가지, 칼슘 흡수율 높이는 방법" (키워드+숫자+가치)

**헤딩 구조 개선:**
- H2만 사용 → H2+H3 계층 구조 도입
- 포커스 키워드를 H2에 최소 1회 포함

**내부 링크 확대:**
- 현재 1개 → 최소 3개 관련 포스트 링크
- 결론부에 "함께 읽으면 좋은 글" 섹션 추가

**이미지 최적화:**
- 본문에 관련 이미지 2~3개 추가
- 모든 이미지에 키워드 포함 alt 태그

### [중간] 4순위: 카테고리 구조 정비
**현재:** 카테고리 네비게이션 부재, 태그만 30개+
**개선안:**
1. 건강정보 (영양소, 식단, 운동)
2. 생활꿀팁 (생활습관, 계절관리)
3. 뷰티/스킨케어 (피부, 헤어, 바디)
4. 식품/영양 (건강식품, 요리, 레시피)
5. 정부지원/복지 (지원금, 제도, 혜택)

### [중간] 5순위: 구조화 데이터 강화
- Article 스키마 → 모든 포스트 적용
- FAQPage 스키마 → Q&A 형식 포스트
- HowTo 스키마 → 방법론 포스트
- BreadcrumbList → 카테고리 경로
