# SEO 감사 스킬

## 온페이지 SEO 체크리스트

### 제목 태그 (Title Tag)
- [ ] 포커스 키워드가 제목 앞쪽에 위치
- [ ] 40~60자 이내
- [ ] 클릭을 유도하는 감성 요소 포함
- [ ] 중복 제목 없음

### 메타 설명 (Meta Description)
- [ ] 155자 이내
- [ ] 포커스 키워드 포함
- [ ] 행동 유도 문구(CTA) 포함
- [ ] 각 포스트별 고유한 설명

### 헤딩 구조 (Heading Structure)
- [ ] H1: 1개만 사용 (=글 제목)
- [ ] H2: 5~10개 (주요 섹션)
- [ ] H3: 필요시 하위 섹션
- [ ] 포커스 키워드 H2에 최소 1회 포함
- [ ] 계층 구조 준수 (H1→H2→H3, H2→H4 금지)

### URL/슬러그
- [ ] 짧고 의미 있는 URL
- [ ] 포커스 키워드 포함
- [ ] 불필요한 숫자/날짜 제거

### 이미지 최적화
- [ ] alt 텍스트에 키워드 포함
- [ ] 파일명 의미 있게 설정
- [ ] WebP/최적화 포맷 사용
- [ ] 적절한 크기 (너비 800px 이내)

### 내부/외부 링크
- [ ] 내부 링크 최소 2개
- [ ] 외부 링크 1~2개 (권위 있는 출처)
- [ ] 깨진 링크 없음
- [ ] 앵커 텍스트 다양화

### 콘텐츠 품질
- [ ] 최소 1,500자
- [ ] 독창적 콘텐츠
- [ ] 키워드 밀도 1~2%
- [ ] 가독성 높은 문단 구조
- [ ] 멀티미디어 활용 (이미지, 표, 리스트)

## 기술적 SEO

### 구조화 데이터
- Article 스키마 적용
- BreadcrumbList 스키마 (카테고리 경로)
- FAQ 스키마 (질문형 포스트)

### 페이지 속도
- 이미지 lazy loading
- CSS/JS 최소화
- 불필요한 외부 스크립트 제거

### 모바일 최적화
- 반응형 디자인 확인
- 터치 타겟 크기 (최소 48px)
- 가로 스크롤 없음

## 점수 산출 (100점)
| 항목 | 배점 | 기준 |
|------|------|------|
| 제목/메타 | 20점 | 키워드 포함, 길이 적절 |
| 헤딩 구조 | 15점 | 계층 준수, 키워드 포함 |
| 콘텐츠 품질 | 25점 | 길이, 독창성, 키워드 밀도 |
| 링크 전략 | 15점 | 내부/외부 링크 적절 |
| 이미지 최적화 | 10점 | alt, 크기, 포맷 |
| 기술적 SEO | 15점 | 스키마, 속도, 모바일 |
