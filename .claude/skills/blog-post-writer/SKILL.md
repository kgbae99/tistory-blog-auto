# 블로그 포스트 작성 스킬

## 포스트 구조 템플릿

### 1. 제목 (Title)
- 40~60자, 포커스 키워드 포함
- 숫자/리스트형 선호 ("~하는 5가지 방법", "~TOP 7")
- 감성 hook 추가 ("놓치면 후회할", "전문가가 추천하는")

### 2. 도입부 (Intro) — 200~300자
- 독자의 문제/궁금증 공감
- 이 글에서 얻을 수 있는 가치 명시
- 포커스 키워드 자연스럽게 포함

### 3. 본문 (Body) — H2 5~8개
각 H2 섹션:
- 300~500자
- 실용적 정보 제공
- 불릿 포인트/번호 리스트 활용
- 필요시 H3 하위 섹션

### 4. 쿠팡 상품 삽입 (3개)
- 2번째, 4번째, 6번째 H2 뒤에 배치
- 상품명 + 가격 + 별점 + 구매 링크
- "이 제품이 도움이 될 수 있어요" 같은 자연스러운 전환문

### 5. 애드센스 광고 위치
- 첫 번째 H2 뒤
- 세 번째 H2 뒤
- 결론 직전

### 6. 결론 (Conclusion) — 200~300자
- 핵심 내용 요약
- 행동 촉구 (CTA)
- 관련 포스트 링크 2개

### 7. 법적 고지
```
이 포스팅은 쿠팡 파트너스 활동의 일환으로, 이에 따른 일정액의 수수료를 제공받습니다.
본 콘텐츠는 정보 제공 목적이며, 의학적 조언을 대체하지 않습니다. 건강 관련 결정은 전문의와 상담하세요.
```

### 8. 태그
- 포커스 키워드 + 관련 키워드 5~10개

## 문체 가이드
- ~입니다/~합니다 (존댓말)
- 전문 용어는 쉽게 풀어서 설명
- "건강온도사가 알려드리는" 같은 브랜딩 문구 자연스럽게 삽입
- 과장/허위 정보 절대 금지
- "~라고 합니다" 대신 구체적 출처 명시
