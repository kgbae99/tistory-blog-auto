# 키워드 리서치 스킬

## 리서치 프로세스

### Step 1: 시드 키워드 도출
- 블로그 주제(건강/생활/뷰티)에서 시드 키워드 5~10개 선정
- 계절성 반영 (현재 월 + 다음 달 키워드)

### Step 2: 키워드 확장
- 시드 키워드 + 수식어 조합
  - "~방법", "~추천", "~효과", "~부작용", "~가격", "~후기"
  - "2026", "최신", "인기", "TOP", "비교"
- 질문형 키워드: "~하면 어떻게 되나요", "~에 좋은 것"

### Step 3: 검색량/경쟁도 분석
WebSearch로 다음을 확인:
- Google: `site:tistory.com "{키워드}"` → 경쟁 포스트 수
- 네이버: 자동완성 + 연관검색어 확인
- 상위 노출 포스트의 품질/길이 확인

### Step 4: 키워드 분류
| 분류 | 설명 | 예시 |
|------|------|------|
| 정보성 | 지식/방법 탐색 | "비타민D 효능" |
| 상업성 | 구매 의도 | "비타민D 추천 제품" |
| 비교성 | 제품/방법 비교 | "비타민D vs 비타민C" |
| 리뷰성 | 후기/경험 | "비타민D 3개월 복용 후기" |

### Step 5: 최종 키워드 선정 기준
1. 월간 검색량 1,000+ (추정)
2. 경쟁도 중~낮음
3. 쿠팡 상품 매칭 가능
4. 블로그 기존 콘텐츠와 시너지
5. 상업적 의도 포함
