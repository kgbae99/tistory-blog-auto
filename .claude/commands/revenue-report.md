# /revenue-report — 수익 현황 리포트

## 분석 항목

### 1. 콘텐츠 성과 분석
기존 포스트를 분석하여:
- 최근 발행 포스트 목록
- 발행 빈도 및 카테고리 분포
- 제목/주제 패턴 분석

### 2. 쿠팡 파트너스 수익 분석
- 상품 링크 삽입된 포스트 확인
- 카테고리별 추천 상품 전략 평가
- 전환율 높은 상품 유형 분석

### 3. 애드센스 최적화 분석
블로그 HTML을 검토하여:
- 광고 배치 현황
- 광고 밀도 적절성
- 광고 포맷 최적화 여지

### 4. SEO 성과 분석
- 구조화 데이터 적용 현황
- 메타 태그 최적화 상태
- 내부 링크 구조

### 5. 개선 전략 제안
- 수익화 효율 높은 키워드 추천
- 쿠팡 상품 카테고리 조정
- 광고 배치 변경 제안
- 포스팅 빈도 최적화
