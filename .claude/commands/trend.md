# /trend — 트렌드 키워드 분석

## 실행 절차

### 1. 실시간 트렌드 수집
WebSearch를 활용하여:
- Google Trends Korea 인기 검색어
- 네이버 실시간 검색어 트렌드
- 건강/뷰티/생활 카테고리 필터

### 2. 키워드 분석
keyword-research 스킬 활용:
- 검색량 추정
- 경쟁도 평가
- 쿠팡 상품 매칭 가능성

### 3. 계절성 키워드 체크
config/settings.yaml의 계절 키워드 달력 참조:
- 현재 시즌 키워드 확인
- 다음 시즌 선점 키워드 추천

### 4. 주간 콘텐츠 캘린더 생성
7개 포스트 주제 추천:
- 월~금: 트렌드 키워드 기반
- 토: 리뷰/비교 포스트
- 일: 시즈널/에버그린 콘텐츠

## 출력 형식
```
[트렌드 분석 리포트] YYYY-MM-DD

🔥 오늘의 핫 키워드 (건강/생활/뷰티)
1. 키워드A — 검색량: 높음, 경쟁: 낮음 ★★★
2. 키워드B — 검색량: 중간, 경쟁: 중간 ★★
3. 키워드C — 검색량: 높음, 경쟁: 높음 ★

📅 이번 주 추천 주제
- 월: [주제] — 타겟 키워드: xxx
- 화: [주제] — 타겟 키워드: xxx
...

🌸 다가오는 시즈널 키워드
- [키워드] — 피크: O월, 지금 준비 시작 권장
```

## 사용법
```
/trend                    # 오늘의 트렌드 분석
/trend "건강식품"          # 특정 분야 트렌드
/trend --weekly           # 주간 캘린더 생성
```
